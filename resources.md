# Resources — curated, not a firehose

Only things worth your time. Use these to go deeper *after* a session, never as a substitute for building.

## The two that matter most
- **Automate the Boring Stuff with Python** — automatetheboringstuff.com — free online, written for non-programmers, project-first. Your spiritual home for Phases 1–8.
- **Official Python Tutorial** — docs.python.org/3/tutorial — the source of truth. Dry but accurate; use to settle "how does X really work".

## Practice / drills
- **Exercism (Python track)** — exercism.org/tracks/python — bite-size exercises with mentor feedback, free.
- **HackerRank / LeetCode (Easy)** — for interview-style drills once you hit Phase 4+.
- **Codewars** — gamified katas, good for the geek streak.

## When you get stuck
- **Python docs** — docs.python.org/3 — the standard library reference.
- **Real Python** — realpython.com — clear tutorials on specific topics (pandas, OOP, venvs).
- **Stack Overflow** — you'll land here from Google; learn to read accepted answers critically.

## Phase-specific
- **pandas (Phase 7):** "10 minutes to pandas" (official) + Real Python's pandas intro.
- **Tooling (Phase 8):** Real Python on virtualenv, pip, and the VS Code debugger; the Pro Git book (free) for git basics.
- **Streamlit (Phase 9 quest):** docs.streamlit.io — dashboards in pure Python, no web skills needed.
