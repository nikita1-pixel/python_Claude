## Key Bugs You Hit (Remember These)
1. **Storing f-string instead of dict** — can't do math or filter on a string later
2. **Nested loops** instead of sequential — each loop should do one job
3. **`total_spent` used before defined** — always define variables before loops
4. **`for` loop outside `else`** — indentation matters in Python
5. **Amount stored as string** — always `float(amount)` when storing numbers
6. **Copilot interference** — accepted code you didn't understand. Turn it off while learning.
